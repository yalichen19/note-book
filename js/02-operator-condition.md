# 运算符
## 算数运算符 + - * / % ++ -- += -= /= *= %= ()
交换a b 
```js
0 / 0 // NaN

a = a + b
b = a - b
a = a - b
```

## 比较运算符 > < <= >= == != 比较结果为 boolean
```js
console.log(undefined == undefined) // true
console.log(Infinity == Infinity) // true
console.log(NaN == NaN) // false
```

## ASC码
一开始由7位二进制数表示，共有128个（0-127）值，字符串比较的就是ASC码的大小

ASCII码: 8位二进制 0-255

## 逻辑运算符 && || ! 运算结果为参与运算成员的值
```js
undefined null 0 '' false // false
```
